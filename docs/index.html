Python process utilities (psutil)

Summary

psutil is a module providing convenience functions for managing processes in a portable way by using Python.

http://code.google.com/p/psutil/

Example usages

>>> import psutil  
>>> p = psutil.Process(3084)  
>>> p.name  
'python'  
>>> p.path  
'/usr/bin'  
>>> p.cmdline  
['/usr/bin/python', '-O']
>>> psutil.test()
