API reference

class psutil.Process(pid)

A class which represents an OS process.
  • pid
    The process pid.
  • name
    The process name.
  • path
    The process path.
  • cmdline
    The command line process has been called with.
  • kill([sig])
    Kill the current process by using signal sig (defaults to SIGKILL).

function get_process_list()

Return a list of Process class instances for all running processes on the local machine.
